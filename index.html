Projects

This is where you will find a range of projects that I have done.

OpenVPN Server

This is an Ubuntu 16.04 Server with OpenSSH and also OpenVPN. This server has both custom made certificates for both clients and also for the VPN to operate properly. This VPN can be accessed from outside the actual network, allowing others from all across the world to access this VPN. I created certifications for all users (including all mobile operating systems such as IOS and Jellybean).
Contributors: Ahmeen Muhammad

Port Scanner with GUI Interface

We used knowledge of Python Fundamentals as well as the Tkinter and Sockets libraries in Python to construct a GUI interfaced port scanner. The scanner takes a target IP address while on a network and scans ports on the specified target in order to determine and output which ports are open and closed, and then saves this data to a .txt file for the user's convenience. By creating this, we allow the user another option in trying to complete the reconnaissance phase of the Penetration Testing hierarchy.
Contributors: Ahmeen Muhammad, Heidee Smith
